// Smart Route Navigator //
Smart Route Navigator is a Java application developed to explore shortest path algorithms using graph data structures.

The project implements Dijkstra's Algorithm, A* Search, and Breadth-First Search (BFS) to compute routes between two nodes and compare their performance on different graph types.

This project was built to strengthen my understanding of graph algorithms, data structures, Java collections, and object-oriented programming.

--> Features

- Shortest path calculation
- Dijkstra's Algorithm
- A* Search
- Breadth First Search
- Weighted graph support
- Unweighted graph support
- Performance benchmarking
- Large graph testing
- Path visualization
- Edge case handling



--> Technologies

- Java
- OOP
- Graph Data Structure
- Java Collections Framework
